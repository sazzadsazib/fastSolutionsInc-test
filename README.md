## Fast Pay Website

To run this project,

```$xslt
$ git clone https://github.com/sazzadsazib/fastpay-test.git
```

Install Dependency

```$xslt
yarn install
```

Run the project

```$xslt
yarn start
```

Add .env

```$xslt
touch .env
```

Provide env information

```$xslt
REACT_APP_SECRET_KEY=your-key
```
